# Nakazasen AI Router 0.2.1

## Highlights

- General-purpose AI capacity layer positioning, no translation-first bias.
- Live provider conformance matrix with sanitized opt-in reports.
- Segmentation and aggregation primitives for long payloads.
- Persistent SQLite job queue for durable worker loops.
- Metrics and observability snapshots for router and job state.
- Provider quota/capacity policy primitives and fallback sorting.
- SDK-first external integration hardening with wheel install smoke.
- Static local dashboard prototype for JSON metrics.

## Install

```powershell
pip install git+https://github.com/Nakazasen/nakazasen-ai-router.git@v0.2.1
```

## SDK-first integration

Use `nakazasen_ai_router` directly from Python apps to route, segment, queue, throttle, and observe AI workloads without running a Router Server.

## Safety

Live provider tests remain opt-in. Reports, metrics, job metadata, and dashboard samples avoid raw API keys, prompts, authorization headers, and provider responses.

## Verification

Release verification includes the offline test suite, package build, docs/mojibake checks, and install-from-wheel smoke.
